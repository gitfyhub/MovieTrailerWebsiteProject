1. Ensure you have python installed on your machine.  If you do not going to the internet and download Python 2.7.9.

2. Copy all of these files into a directory
	media.py
	fresh_tomatoes.py
	entertainment_center.py
	
3. Open entertainment_center.py in IDLE, then click Run, then select Run Module.

A browser window will open that contains fresh_tomatoes.html which displays the movie trailers page.
